# SeekO

SeekO is a home-education application for parents to keep their kids' brain juices busy. SeekO to enjoy the following benefits: to build your children's knowledge, critical thinking and creativity skills and ensure completeness in education with dedicated categories like Fun Experiments, Did you know, Bedtime Stories and Quizzes etc.
